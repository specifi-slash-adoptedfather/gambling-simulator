# RELEASE_CHECKLIST

## 目标

用于微信小程序提审 / 上线前的最终检查。

当前项目定位：
- 朋友临时打赌记账工具
- 本地多赌局管理
- 手动关闭下注
- 手动出结果
- 自动结算
- 导出海报
