# SESSION_NOTES

## 当前项目目标

做一个可上传到微信小程序的「打赌模拟器」前端，核心体验包括：
- 编辑赌局标题
- 维护正方 / 反方下注列表
- 自动计算总奖池、赔率、小计
- 保存本地数据
- 导出当前赌局为图片

## 当前技术栈

- `uni-app`
- `Vue 3`
- `Vite`
- Node 内置测试器 `node:test`

## 重要实现说明

### 图片导出策略
当前“保存为图片”不是页面截图，而是：
1. 读取当前页面数据
2. 用隐藏 `canvas` 重绘海报
3. 导出 png
4. 微信小程序保存到相册

优点：
- 更稳定
- 更适合微信小程序
- 不依赖 H5 截图库

### 编码注意事项
这个项目之前出现过中文乱码 / BOM 导致的异常。
后续修改时注意：
- 文件尽量保持 UTF-8 无 BOM
- 如果中文再次出现异常，优先检查文件编码
- 页面中文文案可继续集中在常量对象中维护
