# 打赌模拟器 - uni-app 版本

这是一个可编译到微信小程序的打赌模拟器前端，当前基于 `uni-app + Vue 3 + Vite`。

当前已经支持：
- 编辑标题
- 维护正方 / 反方下注列表
- 自动计算总奖池、赔率、小计
- 本地持久化保存
- 生成并保存图片海报（微信小程序保存到相册，H5 预览图片）

## 项目结构

```text
src/
├─ pages/
│  └─ index/
│     └─ index.vue      # 主页面
├─ utils/
│  └─ bet.js            # 下注数据清洗与计算逻辑
├─ App.vue
├─ main.js
├─ manifest.json
└─ pages.json
```

## 运行命令

```bash
# H5 开发
npm run dev:h5

# 微信小程序开发
npm run dev:mp-weixin

# H5 构建
npm run build:h5

# 微信小程序构建
npm run build:mp-weixin
```

## 微信小程序调试

1. 执行：`npm run build:mp-weixin`
2. 打开微信开发者工具
3. 导入目录：`dist/build/mp-weixin`
4. 在页面中增删下注、修改标题、测试保存图片

## 保存为图片说明

当前不是对页面做截图，而是：
- 根据当前数据在隐藏 `canvas` 上重绘一张海报
- 导出为 PNG
- 在微信小程序中调用保存到相册

这样做的优点：
- 更稳定
- 不依赖 DOM 截图库
- 更适合微信小程序环境

相关代码位置：
- 页面入口：`src/pages/index/index.vue`
- 海报绘制：`src/pages/index/index.vue`
- 数据工具：`src/utils/bet.js`
